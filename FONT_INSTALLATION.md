# วิธีติดตั้งฟอนต์ภาษาไทยสำหรับ PDF

## ปัญหา
แอปไม่สามารถดาวน์โหลดฟอนต์ภาษาไทยจาก URL อัตโนมัติได้

## วิธีแก้ไข (แนะนำ)

### ขั้นตอนที่ 1: ดาวน์โหลดฟอนต์
1. เปิดเบราว์เซอร์และไปที่: https://fonts.google.com/specimen/Noto+Sans+Thai
2. คลิก "Download family" เพื่อดาวน์โหลดไฟล์ ZIP
3. แตกไฟล์ ZIP และหาไฟล์ `NotoSansThai-Regular.ttf`

### ขั้นตอนที่ 2: วางไฟล์ในโปรเจกต์
1. ไปที่โฟลเดอร์ `C:\src_flutter\hummans\assets\fonts\`
2. **ลบโฟลเดอร์ `NotoSansThai-Regular.ttf` ที่มีอยู่** (ถ้ามี)
3. **วางไฟล์ `NotoSansThai-Regular.ttf`** (ไฟล์จริง ไม่ใช่โฟลเดอร์) ลงในโฟลเดอร์นี้

### ขั้นตอนที่ 3: รันคำสั่ง
```bash
flutter pub get
```

### ขั้นตอนที่ 4: ทดสอบ
ลองดาวน์โหลดเอกสารอีกครั้ง ฟอนต์ภาษาไทยควรทำงานได้แล้ว

## หมายเหตุ
- ไฟล์ฟอนต์ควรมีขนาดประมาณ 200-300 KB
- ต้องเป็นไฟล์ `.ttf` จริงๆ ไม่ใช่โฟลเดอร์
- หลังจากวางไฟล์แล้ว ต้องรัน `flutter pub get` เพื่อให้ Flutter เห็นไฟล์ใหม่
